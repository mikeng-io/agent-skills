# Creative Domains

Reference list of creative domains for deep-* skill expert selection.

## Domains

### design
```yaml
name: design
trigger_signals:
  - design
  - visual
  - layout
  - typography
  - color
  - spacing
  - hierarchy
  - figma
  - sketch
  - component
  - design system
  - UI
  - interface
  - wireframe
  - mockup
expert_role: "Visual Designer"
focus_areas:
  - Visual hierarchy and information architecture
  - Typography scale and readability
  - Color system consistency
  - Spacing and grid alignment
  - Component reusability
  - Design system adherence
  - Cross-platform consistency
standards:
  - Material Design 3
  - Apple Human Interface Guidelines
  - Web Content Accessibility Guidelines (visual)
  - Gestalt principles
```

### ux
```yaml
name: ux
trigger_signals:
  - UX
  - user experience
  - user flow
  - user journey
  - usability
  - onboarding
  - error state
  - empty state
  - task completion
  - interaction
  - prototype
  - testing
  - friction
  - mental model
expert_role: "UX Researcher"
focus_areas:
  - User flow clarity and efficiency
  - Task completion path optimization
  - Error recovery experience
  - Onboarding friction reduction
  - Mental model alignment
  - WTF-moment detection (obvious confusion points)
  - Loading and empty state handling
standards:
  - Nielsen's 10 Usability Heuristics
  - Fitts's Law
  - Cognitive load theory
  - User-centered design (ISO 9241-210)
```
